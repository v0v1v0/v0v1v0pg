Usage
GSGeoPackageDataStore$new(
  xml = NULL,
  name = NULL,
  description = "",
  enabled = TRUE,
  database = NULL
)
Arguments
xml

an object of class xml_node-class to create object from XML

name

coverage store name

description

coverage store description

enabled

whether the store should be enabled or not. Default is TRUE

database

database
