theme_map R Documentation

Map Theme

Description

Theme for maps which uses the 'Times' family and has a transparent background.

Usage

theme_map(...)

Arguments

...

additional parameters

Value

ggplot2 theme

Examples

library(ggplot2)
data(oregon)

ggplot(oregon, aes(group = county)) +
    geom_district() +
    scale_fill_penn82() +
    theme_map()
