MouseAll R Documentation

All GO-terms for Mouse

Description

A data set with all the GO-terms for Mouse.

Usage

MouseAll

Format

A data frame with 337505 rows and 2 variables:

entrezgene_id

The entrezgene_id.

go_id

The respective GO-terms.
