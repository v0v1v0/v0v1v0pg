PlotSimulation R Documentation

Plot simulated data

Description

The input data is usually created by SimulateKinetics

Usage

PlotSimulation(
  sim.df,
  ntr = TRUE,
  old = TRUE,
  new = TRUE,
  total = TRUE,
  line.size = 1
)

Arguments

sim.df

the input data frame
ntr

show the ntr?
old

show old RNA?
new

show new RNA?
total

show total RNA?
line.size

which line size to use

Value

a ggplot object

See Also

SimulateKinetics for creating the input data frame

Examples

PlotSimulation(SimulateKinetics(hl=2))
