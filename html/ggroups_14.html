qmat R Documentation

Matrix Q

Description

Creates the genetic group contribution matrix.

Usage

qmat(ped2)

Arguments

ped2

: The output data.frame from gghead (for more details: ?gghead)

Value

Q matrix

Examples

ped = data.frame(ID=c(3,4,6,5), SIRE=c(1,3,4,1), DAM=c(2,2,5,2))
ped2 = gghead(ped)
qmat(ped2)
