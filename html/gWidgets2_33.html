gimage R Documentation

A widget for displaying an image file

Description

A widget for displaying an image file

generic for toolkit dispatch

Usage

gimage(
  filename = "",
  dirname = "",
  stock.id = NULL,
  size = "",
  handler = NULL,
  action = NULL,
  container = NULL,
  ...,
  toolkit = guiToolkit()
)

.gimage(
  toolkit,
  filename = "",
  dirname = "",
  stock.id = NULL,
  size = "",
  handler = NULL,
  action = NULL,
  container = NULL,
  ...
)

Arguments

filename

basename of file
dirname

dirname of file
stock.id

stock id of icon (if non NULL)
size

size of icon when a stock id (toolkit dependent)
handler

handler if image is clicked on.
action

passed to handler
container

parent container
...

passed to add method of parent
toolkit

toolkit
