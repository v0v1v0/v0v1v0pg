fun.poly R Documentation

Performs polynomial to use in GMDH Combinatorial.

Description

Performs polynomial to use in GMDH Combinatorial.

Usage

fun.poly(X, G = 2)

Arguments

X

matrix containing independent variables in the model.
G

polynomial. Default is 2, original Ivakhnenko polynomial. 0: normal regression. 1: only interaction terms.

Value

A matrix with n columns corresponding with Ivakhnenko polynomial
