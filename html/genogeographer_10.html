main_alleles R Documentation

AIMs markers in Precision ID Ancestry Panel (Thermo Fisher Scientific)

Description

List of markers with their main and alternative allele. The markers is the union of Seldin's and Kidd's markers.

Usage

main_alleles

Format

List of 164 markers

locus

Locus/Marker names

main_allele

The main allele (alleles are in lexicographic order)

other_allele

The other variant
