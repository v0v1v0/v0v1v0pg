coef.bic.BLMCP R Documentation

Extract coefficients from a "bic.BLMCP" object

Description

This function extracts the coefficients of main effects and interactions from a BIC BLMCP model, using the stored "bic.BLMCP" object.

Usage

## S3 method for class 'bic.BLMCP'
coef(object, ...)

Arguments

object

Fitted "bic.BLMCP" model object.
...

Not used. Other arguments to get coefficients.

Value

The object returned depends on the ... argument which is passed on to the coef method for bic.BLMCP objects.

alpha

The matrix of the coefficients for main environmental effects.
beta

The matrix of the regression coefficients for all main genetic effects (the first row) and interactions.
