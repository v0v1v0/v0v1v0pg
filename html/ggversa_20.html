Pop_PR R Documentation

Pop_PR

Description

El archivo contiene el tamaño poblacional de Puerto Rico en 2015

Usage

data(Pop_PR)

Format

El archivo incluye 3 columnas con 37 filas

Edad

El cohorte de edad de la poblacion

Sexo

Variable discreta para Mujer y Hombre

Valor

El tamaño poblacional del cohorte

Datos de Censo de EEUU de 2015.

Examples

head(Pop_PR)
