epil2 R Documentation

Seizure Counts for Epileptics - Extended

Description

Extended version of the epil dataset of the MASS package. The three transformed variables Visit, Base, and Age used by Booth et al. (2003) have been added to epil.

Usage

epil2

Format

A data frame with 236 observations on the following 12 variables:

y

an integer vector.

trt

a factor with levels "placebo" and "progabide".

base

an integer vector.

age

an integer vector.

V4

an integer vector.

subject

an integer vector.

period

an integer vector.

lbase

a numeric vector.

lage

a numeric vector.

Visit

(rep(1:4,59) - 2.5) / 5.

Base

log(base/4).

Age

log(age).

References

Booth, J.G., G. Casella, H. Friedl, and J.P. Hobert. (2003) Negative binomial loglinear mixed models. Statistical Modelling 3, 179â€“191.
