msp.groups R Documentation

A List of Human Multi-subunit Proteins

Description

A list of genes encoding 82 multi-subunit proteins were extracted from human genes.

Usage

data(msp.groups)

Format

A List of 82 character vectors.

Details

A multi-subunit protein is named in the format of Multi.XXX where XXX is a member gene picked arbitrarily from the subunit gene group. Note this naming rule is just for convenience.

References

Zhaoyuan Fang, Weidong Tian and Hongbin Ji. A Network-Based Gene Weighting Approach for Pathway Analysis. Submitted.

Examples

data(msp.groups)
