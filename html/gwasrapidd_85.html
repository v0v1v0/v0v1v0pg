is_empty_str R Documentation

Is string empty or an all whitespace string?

Description

Matches the string vector against "^\\s*$".

Usage

is_empty_str(str, convert_NA_to_FALSE = TRUE)

Arguments

str

A character vector.
convert_NA_to_FALSE

Whether to treat NA as NA (convert_NA_to_FALSE = FALSE) or whether to return FALSE when an NA is found (convert_NA_to_FALSE = TRUE).

Value

A logical.
