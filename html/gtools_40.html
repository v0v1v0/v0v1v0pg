Randomly Permute the Elements of a Vector

Description

Randomly Permute the elements of a vector

Usage

permute(x)

Arguments

x

Vector of items to be permuted

Details

This is simply a wrapper function for sample.

Value

Vector with the original items reordered.

Author(s)

Gregory R. Warnes greg@warnes.net

sample

Examples


x <- 1:10
permute(x)
