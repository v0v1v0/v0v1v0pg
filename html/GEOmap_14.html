demcmap R Documentation

Color Map from DEM

Description

create a color map from a DEM (Digital Elevation Map)

Usage

demcmap(ZTOPO, n = 100, ccol = NULL)

Arguments

ZTOPO

Topography structure
n

number of colors
ccol

color structure

Value

vector of rgb colors

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>
