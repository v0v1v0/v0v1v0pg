getId

Description

Extract the unique 16 digits identification from 'Lattes' XML file.

Usage

getId(curriculo)

Arguments

curriculo

'Lattes' XML imported as 'xml2::read_xml()'.

Details

Curriculum without this information will return NULL.

Value

character
