gtm_zones_update R Documentation

Update an existing zone

Description

This takes a zone resource and overwrites the existing zone in GTM

Usage

gtm_zones_update(account_id, container_id, workspace_id, zone_id, zone_object)

Arguments

account_id

Account Id
container_id

Container Id
workspace_id

Workspace Id
zone_id

Zone Id
zone_object

Zone Object
