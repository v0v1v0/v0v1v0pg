ungroup_ R Documentation

Run ungroup() and as.data.frame()

Description

Run ungroup() and as.data.frame()

Usage

ungroup_(.data)

Arguments

.data

grouped data.frame

Value

an ungrouped data.frame()

Examples


data <- data.frame(m=c(1,2), n=c(2,3), group=c('a','b'))
data %>%
    group_by(group) %>%
    ungroup_()
