GeomFromPath R Documentation

ggpath: Robust Image Rendering Support for 'ggplot2'

Description

A 'ggplot2' extension that enables robust image grobs in panels and theme elements.

Usage

GeomFromPath

GeomRefLines

Format

An object of class GeomFromPath (inherits from Geom, ggproto, gg) of length 5.

An object of class GeomRefLines (inherits from Geom, ggproto, gg) of length 5.
