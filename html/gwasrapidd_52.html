get_studies_by_full_pvalue_set R Documentation

Get GWAS Catalog studies by full summary statistics criterion

Description

Gets studies that either have full summary statistics or studies that do not have it.

Usage

get_studies_by_full_pvalue_set(
  full_pvalue_set = NULL,
  verbose = FALSE,
  warnings = TRUE,
  page_size = 20L
)

Arguments

full_pvalue_set

A logical (scalar!) indicating to retrieve studies with full summary statistics (TRUE) or studies without it (FALSE).
verbose

Whether the function should be verbose about the different queries or not.
warnings

Whether to print warnings.
page_size

An integer scalar indicating the page value to be used in the JSON requests, can be between 1 and 1000.

Value

A studies object.
