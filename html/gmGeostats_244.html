spectralcolors R Documentation

Spectral colors palette based on the RColorBrewer::brewer.pal(11,"Spectral")

Description

Spectral colors palette based on the RColorBrewer::brewer.pal(11,"Spectral")

Usage

spectralcolors(n)

Arguments

n

number of colors

Value

a palette, i.e. a list of colors, from dark blue to dark red over pale yellow.

Examples

(cls=spectralcolors(20))
