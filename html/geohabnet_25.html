gplot R Documentation

Plot a Raster* object

Description

This is a wrapper for terra::plot()

Usage

gplot(x, ...)

Arguments

x

a Raster* object
...

additional arguments passed to terra::plot()

Value

a plot

Examples

r <- terra::rast(nrows=108, ncols=21, xmin=0, xmax=10)
gplot(r)
gplot(r, col = "red")
gplot(r, col = "red", breaks = 10)
