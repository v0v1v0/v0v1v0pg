get_studies_by_efo_uri R Documentation

Get GWAS Catalog studies by EFO URIs

Description

Gets studies that match EFO URI.

Usage

get_studies_by_efo_uri(
  efo_uri = NULL,
  verbose = FALSE,
  warnings = TRUE,
  page_size = 20L
)

Arguments

efo_uri

A character vector of EFO URIs.
verbose

Whether the function should be verbose about the different queries or not.
warnings

Whether to print warnings.
page_size

An integer scalar indicating the page value to be used in the JSON requests, can be between 1 and 1000.

Value

A studies object.
