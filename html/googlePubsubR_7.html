msg_encode R Documentation

Encode Pub/Sub message

Description

Converts an object into a base64 string

Usage

msg_encode(x)

Arguments

x

A serializeable object

Value

character a base64 encoded string

Examples

## Not run: 
library(jsonlite)

mtcars %>% 
  toJSON(auto_unbox = TRUE) %>%
  msg_encode() %>% 
  PubsubMessage()
