rank_sigmoid

Creates a longer dataframe with coordinates for a smoothed line.

Usage

rank_sigmoid(x, y, smooth = 8, direction = "x")

Arguments

x

vector
y

vector
smooth

smooth parameter. Higher means less smoothing
direction

the character x or y depending of smoothing direction

Value

a data frame
