grob_matrix R Documentation

Grob Matrix

Description

Initialize a grob matrix object, to be used within grob_col.

Usage

grob_matrix(x)

Arguments

x

Either a data.frame, a matrix or a vector.

Details

Learn more in vignette("grob_matrix")

Value

An R6 object of the grob matrix class.

Examples


data.frame(
  v1 = c(15, 4, 16, 11),
  v2 = c(10, 30, 3, 10)
  ) %>%
  grob_matrix() %>%
  view_grob()
