str_metrics R Documentation

Get font metrics for a string.

Description

Get font metrics for a string.

Usage

str_metrics(
  x,
  fontname = "sans",
  fontsize = 12,
  bold = FALSE,
  italic = FALSE,
  fontfile = ""
)

Arguments

x

Character vector of strings to measure
fontname

Font name
fontsize

Font size
bold, italic

Is text bold/italic?
fontfile

Font file

Value

A named numeric vector

Examples

str_metrics("Hello World!")
