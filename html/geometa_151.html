Method new()

Method used to instantiate an ISOCodeListValue. By default, addCodeListAttrs = TRUE, to add codelist atributes to root XML. The parameter addCodeSpaceAttr = TRUE by default, and ignored if the valueof addCodeLisAttrs is set to FALSE. The argument setValue sets the value as node text (defaut is TRUE). The argument setValueDescription allows to force having description set as value, default is FALSE in which case the name will be preferred, and in case no name is provided, code value will be used.
