Scatterplot of observed and imputed data

Description

Scatterplot of observed and imputed data

Usage

xyplot(...)

Arguments

...

Any arguments passed to the function.

Value

The output of mice::xyplot and a message about the ggmice equivalent.

Examples

imp <- mice::mice(mice::nhanes, maxit = 1, printFlag = FALSE)
xyplot(imp, bmi ~ age)
