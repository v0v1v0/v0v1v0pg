GENEAcount R Documentation

Count Steps

Description

Step count is the number of step lengths divided by 2.

Usage

GENEAcount(x, ...)

Arguments

x

vector
...

other arguments to be swallowed

Value

single numeric

Examples

   x1 <- c(20, 15, 10)
   GENEAcount(x1)
   x2 <- c(300, 255, 111)
   GENEAcount(x2)
