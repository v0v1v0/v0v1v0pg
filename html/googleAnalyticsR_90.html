ga_webproperty R Documentation

Get a web property

Description

Gets metadata for one particular web property

Usage

ga_webproperty(accountId, webPropertyId)

Arguments

accountId

Account Id
webPropertyId

Web Property Id e.g. UA-12345-1

Value

webproperty

See Also

Other account structure functions: ga_account_list(), ga_accounts(), ga_view(), ga_view_list(), ga_webproperty_list()

Examples

library(googleAnalyticsR)
ga_auth()
wp <- ga_webproperty(1058095, "UA-1058095-1")
