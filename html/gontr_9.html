BP GO-terms as an Edgelist for Rat

Description

A data set containing BP, MF and CC GO-terms as an edgelist for Rat

Usage

Rat

Format

A list containing 3 objects:

BPlist

BP GO-terms edgelist.

MFlist

MF GO-terms edgelist.

CClist

CC GO-terms edgelist.
