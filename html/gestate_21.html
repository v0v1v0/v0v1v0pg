freedmanpower R Documentation

Calculate Freedman power from number of events

Description

Calculate Freedman power from number of events

Usage

freedmanpower(events, HR, ratio = 1, alpha1 = 0.025)

Arguments

events

Number of events.
HR

Hazard Ratio.
ratio

Randomisation ratio. Default=1.
alpha1

1-sided alpha. Default=0.025.

Value

Power as a decimal
