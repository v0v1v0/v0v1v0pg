as_R6 R Documentation

As R6

Description

Debug method that strips all gqlr classes and assigns the class as 'R6'

Usage

as_R6(x)

Arguments

x

any object. If it inherits 'R6', then the class of x is set to 'R6'

Examples

Int <- getFromNamespace("Int", "gqlr")$clone()
print(Int)
print(as_R6(Int))
