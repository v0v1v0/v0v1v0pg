teeth R Documentation

Add Teeth to line

Description

Add teeth marks to a line.

Usage

teeth(x, y, h, rot, col = "black", border = "black")

Arguments

x

x-coordinates
y

y coordinates
h

height of tooth
rot

rotation of teeth
col

color of line
border

color of border, default= col

Details

The rotation is usually determined by consecutive x-y points

Value

Graphical Side effect

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu

See Also

thrust

 plot(c(-5,5), c(-5,5), asp=1, type='n' )
