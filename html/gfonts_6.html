included_fonts R Documentation

Detail about included fonts.

Description

Id and version of fonts included and available through use_pkg_gfont.

Usage

included_fonts

Format

A data frame with 8 rows and 5 variables:

id

Id for the font.

family

Name of the font.

category

Category.

version

Version number.

lastModified

Last modified date.
