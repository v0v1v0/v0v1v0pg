Umatch R Documentation

Matching of Orthogonal Matrices for Cayley transforms

Description

This function matches an orthogonal matrix to the importance of the columns of the matrix to which it should be matched.

Usage

Umatch(from, to)

Arguments

from

Matrix: orthogonal
to

Matrix: orthogonal

Value

mat

Matched matrix.

Author(s)

Bernhard Pfaff
