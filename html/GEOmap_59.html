goodticdivs R Documentation

Nice tic division

Description

Determine a reasonable tick division for lat-lon tic marks.

Usage

goodticdivs(ddeg)

Arguments

ddeg

degree differnce

Details

Designed to give approximately 4-6 divisions for plotting given the range input.

Value

K

suggested divisor
