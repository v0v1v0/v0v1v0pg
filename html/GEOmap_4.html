along.great R Documentation

Along A great Arc

Description

Calculate points along a great arc

Usage

along.great(phi1, lam0, c, Az)

Arguments

phi1

start lat, radians
lam0

start lon, radians
c

distance, radians
Az

Azimuthal direction, radiansm

Details

All input and output is radians

Value

List:

phi

latitudes, radians
lam

longitudes, radians

Author(s)

Jonathan M. Lees<jonathan.lees@unc.edu>

Examples

lat1 <- 48.856578
lon1 <- 2.351828	

A = along.great(lat1*pi/180, lon1*pi/180, 50*pi/180, -63*pi/180)

lat=A$phi*180/pi
lon = A$lam*180/pi
