Format

R6Class object.

Value

Object of R6Class for modelling an ISO Distance measure

Methods

new(xml,value, uom, useUomURI)

This method is used to instantiate an ISODistance. The uom argument represents the symbol of unit of measure used. The parameter useUomURI can be used to set the uom as URI, its default value is FALSE.
