cube.solid.random R Documentation

Solid Cube

Description

A function to generate a solid cube with random points

Usage

cube.solid.random(p = 3, n = 850 * (2^p))

Arguments

p

dimension of object
n

number of points

Value

points

location of points
edges

edges of the object

Author(s)

Barret Schloerke

References

http://schloerke.github.io/geozoo/cube/

Examples

## Generates a solid cube with random points
cube.solid.random(p = 3, n = 1000)
