ggswissmaps R Documentation

Offers various swiss maps as data frames and 'ggplot2' objects.

Description

Offers various swiss maps as data frames and 'ggplot2' objects and gives the possibility to add layers of data on the maps. Data are publicly available from the swiss federal statistical office. In addition to the maps2 object (a list of 16 swiss maps, at various levels), there are the data frames with the boundaries used to produce these maps (shp_df, a list with 8 data frames).

Details

See the package vignette for some examples (vignette("ggswissmaps_intro", package = "ggswissmaps")).
