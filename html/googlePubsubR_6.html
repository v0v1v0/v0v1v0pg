msg_decode R Documentation

Decode Pub/Sub message

Description

Converts a Pub/Sub message into an object

Usage

msg_decode(x)

Arguments

x

A base64 encoded string

Value

A deserialized object

Examples

## Not run: 
library(jsonlite)

pulled_msgs$receivedMessages$messages$data %>% 
  msg_decode() %>%
  fromJSON()

## End(Not run)
