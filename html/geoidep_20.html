get_sernanp_link R Documentation

Retrieve links to SERNANP for information on natural protected areas.

Description

Retrieve links to SERNANP for information on natural protected areas.

Usage

get_sernanp_link(type = NULL)

Arguments

type

A string. Select only one from the list of available layers, for more information please use 'get_data_sources(provider = "sernanp")'. Defaults to NULL.

Value

A string containing the URL of the requested file.
