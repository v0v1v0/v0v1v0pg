airstrikes_base R Documentation

airstrikes_base

Description

A subset of airstrikes data in Iraq (a subset of airstrikes in 2006) that can be used to construct baseline densities

Usage

airstrikes_base

Format

A tibble with 808 rows and 3 variables:

date

Date

longitude

Longitudes (decimal)

latitude

Latitudees (decimal)

Examples

airstrikes_base
