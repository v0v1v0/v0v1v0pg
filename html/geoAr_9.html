get_bahra R Documentation

Descarga 'base total' de Asentamientos Humanos de la República Argentina (BAHRA)

Description

Descarga 'base total' de Asentamientos Humanos de la República Argentina (BAHRA)

Usage

get_bahra(geo = "ARGENTINA")

Arguments

geo

un character con el nombre del distrito que se quiere descargar (por defecto toda ARGENTINA) Se puede chequear el id con show_arg_codes.

Value

tibble con datos de BAHRA

Examples

get_bahra()
