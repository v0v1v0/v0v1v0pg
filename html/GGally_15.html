find_plot_type R Documentation

Find plot types

Description

Retrieves the type of plot for the specific columns

Usage

find_plot_type(col1Name, col2Name, type1, type2, isAllNa, allowDiag)

Arguments

col1Name

x column name
col2Name

y column name
type1

x column type
type2

y column type
isAllNa

is.na(data)
allowDiag

allow for diag values to be returned
