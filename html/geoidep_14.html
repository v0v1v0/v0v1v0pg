get_inei_link R Documentation

Gets the links to the INEI's basic cartographic information.

Description

Gets the links to the INEI's basic cartographic information.

Usage

get_inei_link(type = NULL)

Arguments

type

A string. Select only one of the following layers; ‘districts’, ‘provinces’, or ‘departments’. Defaults to NULL.

Value

A string containing the URL of the requested file.
