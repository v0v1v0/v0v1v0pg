geo_pretty R Documentation

Pretty print geojson

Description

Pretty print geojson

Usage

geo_pretty(x)

Arguments

x

input, an object of class geojson

Details

Wrapper around prettify

Examples

geo_pretty(point('{ "type": "Point", "coordinates": [100.0, 0.0] }'))

x <- '{ "type": "Polygon",
"coordinates": [
  [ [100.0, 0.0], [100.0, 1.0], [101.0, 1.0], [101.0, 0.0], [100.0, 0.0] ]
  ]
}'
poly <- polygon(x)
geo_pretty(poly)
