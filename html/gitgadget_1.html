add_users_repo R Documentation

Add users to a repo

Description

Add users to a repo

Usage

add_users_repo(
  token,
  repo,
  userfile,
  permission = 20,
  server = "https://gitlab.com/api/v4/"
)

Arguments

token

GitLab token
repo

Repo to update
userfile

A csv file with student information (i.e., username, token, and email)
permission

Permission setting for the repo (default is 20, i.e., reporter)
server

The GitLab API server
