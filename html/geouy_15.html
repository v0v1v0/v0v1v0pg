mvd_barrios_grid R Documentation

Montevideo barrios grid

Description

A dataset containing the cods, names and others attributes as a geofacet grid

Usage

mvd_barrios_grid

Format

A data frame with 62 rows and 4 variables:

name

name of the "Barrio"

code

INE code of the "Barrio"

row

row position in the grid

col

col position in the grid

See Also

Other data: loc_agr_ine, metadata_tables, metadata_wms, metadata, uy_deptos_grid
