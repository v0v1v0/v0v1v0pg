most.freq R Documentation

Return most frequent numbers

Description

get the n most frequent elements in an array

Usage

most.freq(x, n = 1)

Arguments

x

an array of elements
n

integer, default is 1

Value

the most n elements

Examples

most.freq(c('a', 'a', 'b', 'b', 'b', 'c'), n = 2)
most.freq(c(1, 1, 2, 3, 3, 3, 4, 4), n = 2)
