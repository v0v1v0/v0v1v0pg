chow R Documentation

Creating singular Toeplitz lower Hessenberg matrix

Description

returns matrix A = H(alpha) + delta * EYE, such that H[i,j] = alpha^(i-j+1).

Usage

chow(n, alpha = 1, delta = 0)

Arguments

n

order of the matrix
alpha

defaults to 1
delta

defaults to 0

Value

Singular Toeplitz lower Hessenberg matrix
