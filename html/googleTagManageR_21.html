gtm_folders_delete R Documentation

Deletes a GTM Folder

Description

This deletes a folder in the specified workspaces

Usage

gtm_folders_delete(
  account_id,
  container_id,
  workspace_id,
  folder_id,
  force = c("TRUE", "FALSE")
)

Arguments

account_id

Account Id
container_id

Container Id
workspace_id

Workspace Id
folder_id

Folder Id
force

Force deletion without user input

See Also

https://developers.google.com/tag-platform/tag-manager/api/v2/reference/accounts/containers/workspaces/folders/delete
