gmpolyGrow R Documentation

Grow polynomial

Description

Grow a multivariate polynomial by including a new variable.

Usage

gmpolyGrow(pol)

Arguments

pol

a gmpoly object

Value

The "same" multivariate polynomial as pol, except that it has an additional variable.

Examples

library(gmpoly)
pol <- gmpoly("3 x^(1,2) - 1/7 x^(5,3)")
gmpolyGrow(pol)
