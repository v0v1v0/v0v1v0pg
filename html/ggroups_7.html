mat2tab R Documentation

Matrix to tabular

Description

Converts matrix data to tabular data.

Usage

mat2tab(mat)

Arguments

mat

: matrix

Value

data.frame with 2 integer (IDs) and 1 numeric (values) columns.

Examples

ped = data.frame(ID=1:6, SIRE=c(0,0,1,3,1,4), DAM=c(0,0,2,2,2,5))
mat2tab(buildA(ped))
