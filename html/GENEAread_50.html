svm R Documentation

sum vector magnitude

Description

svm acts identically to 'mean.epoch', with the epoch set to the sampling period. In other words, it computes the instantaneous sum of vector magnitudes of the acceleration at each record point. The function takes "AccData", array and vector input. Note that if provided with an array with 4 or more columns, columns 2 to 4 are used â€“ the first column is regard as a timestamp and hence ignored.
