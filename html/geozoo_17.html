Mobius

Description

A function to generate a mobius strip in the third or fourth dimension.

Usage

mobius(p = 3, n = 10000)

Arguments

p

dimension of object. (3)
n

number of points

Author(s)

Barret Schloerke

References

http://schloerke.github.io/geozoo/mobius/mobius/

Examples

## Generates a mobius strip.
mobius(3, n = 1000)
