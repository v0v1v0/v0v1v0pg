gini.1994b R Documentation

Gini Coefficient

Description

To calculate the Gini coefficient according to the following literature: Brown MC. Using Gini-style indices to evaluate the spatial patterns of health practitioners: theoretical considerations and an application based on Alberta data. Soc Sci Med, 1994, 38(9): 1243-1256.

Usage

gini.1994b(x,y)

Arguments

x

vector, population of each region
y

vector, health resource of each region

Value

G

Gini coefficient. Critria: <0.3=best fairness, 0.3-0.4=relative fairness, >0.4=unfairness, >0.6=highly unfair
