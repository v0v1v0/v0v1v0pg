ggperiodic: Easy Plotting of Periodic Data with 'ggplot2'

Description

Implements methods to plot periodic data in any arbitrary range on the fly.

Overview

The only thing you need to do is add the periodic information to a data frame with periodic(). You then can manually wrap your data around any domain with wrap() or just let ggplot2 do it automatically for you
