upstream R Documentation

Upstreamness

Description

Upstreamness

Usage

upstream(x)

Arguments

x

an object of class "decompr" as created using the load_tables_vectors() function from the decompr package.

Examples

# load the decompr package
library(decompr)

# load example data
data(leather)
attach(leather)

# create a leontief decomposed data set
l <- load_tables_vectors(x = inter,
                         y = final,
                         k = countries,
                         i = industries,
                         o = out        )
 
 # apply upstream
 upstream( l )
