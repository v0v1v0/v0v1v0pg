missing_to_na R Documentation

Recursively map all missing values to NA.

Description

Traverse a recursive list and map missing values to NA. Missing values are:

  • NULL

  • "NR"

Usage

missing_to_na(lst, na = NA_character_)

Arguments

lst

A list.

Details

Note: This function is case sensitive on "NR", i.e., it will leave "nr" untouched.

Value

A list whose missing values have been replaced by NA.
