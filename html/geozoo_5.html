cross.polytope R Documentation

Cross Polytope

Description

A function generate a cross polytope, cube dual, with vertices and a wire frame.

Usage

cross.polytope(p = 3)

Arguments

p

dimension of object

Value

points

location of points
edges

edges of the object

Author(s)

Barret Schloerke

References

http://schloerke.github.io/geozoo/cube/

Examples

# Generates a 3-D Cross Polytope
cross.polytope(p = 3)
