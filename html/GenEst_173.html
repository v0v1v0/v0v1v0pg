plot.cpm R Documentation

Plot results of a single CP model

Description

Plot a single cpm model

Usage

## S3 method for class 'cpm'
plot(x, col = "black", ...)

Arguments

x

model of class cpm
col

color to use
...

to be passed down

Examples

  data(wind_RP)
  mod <- cpm(formula_l = l ~ Season, formula_s = s ~ Season,  
           data = wind_RP$CP, left = "LastPresent", right = "FirstAbsent")
 plot(mod)
