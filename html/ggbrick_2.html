Brick row

Usage

brick_row(
  layer,

  brick_height = 1,
  brick_width = 2.5,
  gap = 0.125,
  width = 0.9,
  .geom = "brick"
)

Arguments

layer

Brick layer.
bpl

Number of bricks in the layer.
brick_height

Brick height.
brick_width

Brick width.
gap

Gap between the bricks.
width

Column width
.geom

Geom type for layering. Either 'brick' or 'brick_waffle'
