gf_chain R Documentation

Chain graph

Description

Returns a chain graph of the given size with Long ID type. The vertex IDs are 0, 1, ..., n-1, and the edges are (0, 1), (1, 2), ...., (n-2, n-1).

Usage

gf_chain(sc, n)

Arguments

sc

A Spark connection.
n

Size of the graph to return.

Examples

## Not run: 
gf_chain(sc, 5)

## End(Not run)
